# 系统设置板块（模型接入）

## 一、模型接入查看

进入 53AI KM 系统后，点击左侧导航栏中 “系统设置” 下的 “模型接入” 选项，即可查看当前已接入的各类模型列表，包括硅基流动、DeepSeek、Azure OpenAI、阿里百炼、火山方舟、百度千帆 ModelBuilder 等平台下的不同模型，以及各模型的类型（智能体模型、向量嵌入模型、重排序模型等）。

![image.png](https://kmapitest.53ai.com/api/preview/76a704229e54f0bccd74a4ad1448cdb7.png)

## 二、大模型接入

### （一）选择大模型

1. 1、在 “模型接入” 页面，点击 “添加” 按钮（或相关添加入口），弹出 “选择模型接入” 弹窗。
2. 2、在弹窗中，可选择要接入的大模型平台，如阿里百炼、火山方舟、百度千帆 ModelBuilder、硅基流动、深度求索、OpenAI、Azure OpenAI 等，点击对应平台后的 “添加” 按钮。

   ![90707f89e8462fb64c0254734b084a49.png](https://kmapitest.53ai.com/api/preview/b06ac923e881956d9fb41bdfec219992.png)

### （二）配置大模型

1. 1、以阿里百炼为例，点击 “添加” 后，弹出 “设置阿里百炼” 弹窗。在弹窗中，输入 API Key（为必填项），然后选择要启用的模型，包括逻辑推理模型（如 Qwen-plus、Qwen-Max、Qwen-Flash、Qwen3-Coder-Plus 等）、向量嵌入模型（如 Qwen-text-embedding-v4、Qwen-multimodal-embedding-v1 等）、重排序模型（如 Qwen-gte-rerank-v2 等），通过开关按钮启用或禁用相应模型。
2. 2、以火山方舟为例，点击 “添加” 后，弹出 “设置火山方舟” 弹窗。在弹窗中，输入 API Key（为必填项），然后选择要启用的模型，包括逻辑推理模型（如 Doubao-Seed-1.6、Doubao-Seed-1.6-thinking、Doubao-Seed-1.6-flash、Doubao-1.5-thinking-vision-pro、DeepSeek-R1、DeepSeek-V3 等）、向量嵌入模型（如 Doubao-embedding-vision、Doubao-embedding-large、Doubao-embedding 等），通过开关按钮启用或禁用相应模型。
3. 3、配置完成后，点击 “确定” 按钮，即可完成大模型的接入配置；若不想配置，可点击 “取消” 按钮。
