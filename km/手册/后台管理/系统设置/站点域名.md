# 系统设置板块（站点域名）

## 一、专属域名管理

### （一）查看专属域名

进入 53AI KM 系统后，点击左侧导航栏中 “系统设置” 下的 “站点域名” 选项，在 “专属域名” 区域可查看当前的专属域名，同时支持点击 “复制” 按钮复制域名。

![eaf3c3572ca43e4080c096661994b9dd.png](https://kmapitest.53ai.com/api/preview/c9b6576b6108a660081f6978afd26b35.png)

### （二）设置专属域名

1. 1、在 “专属域名” 区域，点击 “设置” 按钮，弹出 “专属域名” 设置弹窗。
2. 2、在弹窗的输入框中，输入要设置的专属域名前缀（需使用小写字母、数字或 “-”，且不少于 5 个字符），域名后缀为 “.[53ai.com](https://53ai.com/)”。
3. 3、输入完成后，点击 “保存” 按钮，即可完成专属域名设置；若输入不符合要求（如字符不规范、长度不足或域名已被占用），会有相应错误提示，此时可修改后重新尝试，或点击 “取消” 按钮放弃设置。

## 二、独立域名管理

### （一）查看独立域名

在 “站点域名” 页面的 “独立域名” 区域，可查看当前的独立域名，若域名已生效且启用 HTTPS，会有相应标识，同时支持点击 “复制” 按钮复制域名。

### （二）设置独立域名（CNAME 方式）

1. 1、在 “独立域名” 区域，点击 “设置” 按钮，弹出 “独立域名” 设置弹窗。
2. 2、选择 “CNAME 方式”，在输入框中输入独立域名（如 “[www.chat.haitian.com](https://www.chat.haitian.com/)”）。
3. 3、根据提示，从列表中选择合适的 CNAME 地址（如域名在阿里云备案选择 “[agenthub.n1.53ai.com](https://agenthub.n1.53ai.com/)”，在腾讯云备案选择 “[agenthub.n2.53ai.com](https://agenthub.n2.53ai.com/)”，香港服务器选择 “[agenthub.n3.53ai.com](https://agenthub.n3.53ai.com/)”），前往已备案的域名厂商设置域名 DNS 解析。
4. 4、可选择是否启用 HTTPS（默认开启 “启用 HTTPS” 和 “始终使用 HTTPS”），并选择 SSL 证书生成方式：

   * 若选择 “由 53AI 自动生成”，系统会自动生成 SSL 证书。
   * 若选择 “使用自有证书”，需在 “证书” 和 “私钥” 输入框中分别粘贴证书文件内容和证书私钥。
5. 5、设置完成后，点击 “保存” 按钮，完成独立域名（CNAME 方式）的设置；若不想设置，可点击 “取消” 按钮。

   ![9e0fb441cc9b676d5d8417133bec6b4d.png](https://kmapitest.53ai.com/api/preview/4b5d2b6c64ab31d0c8883870995b6e9a.png)

### （三）设置独立域名（自有服务器中转方式）

1. 1、在 “独立域名” 区域，点击 “设置” 按钮，弹出 “独立域名” 设置弹窗。
2. 2、选择 “自有服务器中转” 方式，在输入框中输入独立域名（如 “[www.chat.haitian.com](https://www.chat.haitian.com/)”），若使用子目录，可开启 “使用子目录” 开关并设置子目录（如 “/chat”）。
3. 3、根据提示，可通过两种方式实现：方式一使用 nginx 等 web 服务器做简单配置；方式二使用阿里云、腾讯云、七牛云等云厂商的全站加速服务（DCDN），并记录 Docs Site ID（如 5440233）。
4. 4、设置完成后，点击 “保存” 按钮，完成独立域名（自有服务器中转方式）的设置；若不想设置，可点击 “取消” 按钮。

   ![b2b393369f9da5773ea4046700997d87.png](https://kmapitest.53ai.com/api/preview/0de47359b8db3e09a1d8e25b23bfd372.png)

### （四）删除独立域名

在 “独立域名” 区域，点击 “删除” 按钮，可将已设置的独立域名删除。
