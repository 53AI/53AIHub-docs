## 🚀 快速入门

### 1. 登录后台
访问 https://kmmix.53ai.com/console/?#/index 使用管理员账号登录

### 2. 核心功能概览

| 模块 | 功能 | 用途 |
|------|------|------|
| **应用管理** | AI 产品/工具/提示词/智能体 | 管理前台展示的 AI 应用能力 |
| **站点装修** | Banner/导航/模板 | 自定义站点外观与导航结构 |
| **站点配置** | 平台接入/支付/域名 | 配置站点核心功能与第三方服务 |
| **运营管理** | 用户/订单/订阅 | 管理用户、订单与订阅套餐 |

### 3. 典型工作流程

```mermaid
flowchart TD
    A[平台接入] --> B[添加智能体/AI 工具]
    B --> C[配置站点信息]
    C --> D[设置导航与 Banner]
    D --> E[配置支付方式]
    E --> F[发布站点]
    F --> G[运营管理与数据分析]
```
