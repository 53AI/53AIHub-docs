# 🎨 Banner 图

你可以为站点首页添加**Banner 图**，用于展示品牌视觉、核心产品或活动信息，增强用户第一印象。

![Banner 配置](http://kmdev.53ai.com/api/preview/banner-config.png)

---

## 📦 功能概览

| 功能 | 说明 |
|------|------|
| **图片上传** | 上传 Banner 展示图片 |
| **多图轮播** | 支持多张 Banner 图轮播展示 |
| **轮播设置** | 自定义轮播时间间隔 |
| **排序管理** | 调整 Banner 图显示顺序 |

---

## 📤 上传图片

在此处上传你希望展示的图片，建议上传高质量、符合视觉规范的素材。

### 图片规范

| 要求 | 说明 |
|------|------|
| **推荐尺寸** | 1920×380 像素 |
| **文件大小** | < 10 MB |
| **格式支持** | PNG、JPG、JPEG |
| **数量限制** | 最多 5 张图 |

### 设计建议

**尺寸比例：**
- 使用 1920×380px（约 5:1）获得最佳展示效果
- 确保重要内容在中心区域
- 考虑不同屏幕尺寸的适配

**内容建议：**
- 突出品牌 Logo 和核心信息
- 使用高清图片提升质感
- 文字内容简洁明了
- 配色与站点主题一致

**技术优化：**
- 压缩图片减小文件大小
- 使用 WebP 格式提升加载速度
- 确保图片在不同设备上清晰

## ⏱️ 图片轮播间隔

当 Banner 区存在多张图片时，你可以设置轮播的时间间隔（单位：秒），以控制图片切换节奏，提升浏览体验。

### 推荐设置

| 场景 | 推荐间隔 | 说明 |
|------|----------|------|
| **单图展示** | - | 无需轮播 |
| **2-3 张图** | 5 秒 | 标准轮播速度 |
| **4-5 张图** | 3-4 秒 | 加快轮播节奏 |
| **活动推广** | 6-8 秒 | 给用户更多阅读时间 |

### 设置方法

1. 在「轮播间隔」输入框中输入秒数
2. 建议范围：3-10 秒
3. 点击「保存」生效

## 📋 管理操作

### 添加 Banner

1. 点击「**上传图片**」按钮
2. 选择本地图片文件
3. 等待上传完成
4. 可继续添加更多图片

### 调整顺序

1. 鼠标悬停在图片上
2. 点击拖拽图标
3. 拖动到目标位置
4. 松开鼠标完成排序

### 删除图片

1. 鼠标悬停在图片上
2. 点击右上角删除图标 ❌
3. 确认删除

### 预览效果

1. 配置完成后点击「**预览**」
2. 查看前台实际展示效果
3. 确认无误后点击「**保存**」

## 💡 最佳实践

### 1. 内容规划


### 2. 设计要点

## 📱 响应式适配

Banner 图在不同设备上的展示效果：

| 设备 | 显示宽度 | 建议 |
|------|----------|------|
| **桌面端** | 1920px | 完整展示 |
| **笔记本** | 1366px | 自动裁剪 |
| **平板** | 768px | 居中显示 |
| **手机** | 375px | 自适应缩放 |

**设计建议：**
- 重要内容放在中心安全区域
- 避免在边缘放置关键信息
- 测试不同设备的展示效果

## ❓ 常见问题

### Q1: Banner 图不显示怎么办？
- 检查图片是否上传成功
- 确认图片格式符合要求
- 清除浏览器缓存后刷新

### Q2: 如何临时隐藏 Banner？
- 删除所有 Banner 图即可隐藏
- 或联系技术支持关闭 Banner 模块

### Q3: 支持动态 GIF 吗？
- 当前版本支持 GIF 格式
- 建议文件大小控制在 5MB 以内

### Q4: 可以设置点击跳转吗？
- 当前版本暂不支持
- 如需此功能请联系技术支持
