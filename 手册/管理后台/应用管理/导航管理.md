# 🧭 导航管理

你可以在此处设置站点在前台展示的导航选项，从而使站点信息展示更加条理清晰，最多支持**8 个**导航栏的添加。

![导航管理](http://kmdev.53ai.com/api/preview/navigation-manage.png)

对应前台展示效果：

![前台导航展示](http://kmdev.53ai.com/api/preview/navigation-frontend.png)

---

## 📦 功能概览

| 功能 | 说明 |
|------|------|
| **导航添加** | 创建新的导航栏选项 |
| **类型选择** | 支持系统内置、外部链接、自定义页 |
| **SEO 设置** | 配置标题、关键词、描述 |
| **排序管理** | 调整导航显示顺序 |
| **打开方式** | 设置原窗口或新窗口打开 |

## ➕ 添加导航

进入「**站点装修 → 导航管理**」页面：

- 点击「**添加**」，设置需要的导航栏
- 配置导航栏的类型、名称、跳转地址、打开方式等基础信息，还可进行 SEO 设置

![添加导航](http://kmdev.53ai.com/api/preview/add-navigation.png)

### 基础信息

| 字段 | 说明 | 要求 |
|------|------|------|
| **类型** | 导航跳转类型 | 系统内置/外部链接/自定义页 |
| **名称** | 导航栏显示的文字 | 2-10 字，简洁明了 |
| **跳转地址** | 点击后跳转的 URL | 完整地址或系统路径 |
| **打开方式** | 页面打开位置 | 原窗口/新窗口 |

#### 类型说明

**1. 系统内置**
- 跳转到平台内部功能模块
- 从下拉列表中选择预设页面
- 示例：首页、应用中心、会员中心

**2. 外部链接**
- 跳转到其他网站地址
- 需填写完整 URL（含 https）
- 示例：`https://www.example.com`

**3. 自定义页**
- 跳转到自定义创建的页面
- 需先创建自定义页面
- 示例：活动页、专题页

---

### SEO 设置

配置 SEO 信息有助于搜索引擎收录和展示。

| 字段 | 说明 | 建议 |
|------|------|------|
| **Title** | 页面标题 | 50-60 字符，包含核心关键词 |
| **Keywords** | 页面关键词 | 3-5 个关键词，逗号分隔 |
| **Description** | 页面描述 | 150-160 字符，概括页面内容 |

**SEO 设置示例：**

```
Title: AI 应用中心 - 53AI Hub
Keywords: AI 应用，智能体，AI 工具，人工智能平台
Description: 53AI Hub 应用中心，提供丰富的 AI 应用和智能体，助力企业和个人提升工作效率。
```

## 📋 导航管理

### 编辑导航

1. 找到需要编辑的导航项
2. 点击「**编辑**」按钮
3. 修改相关信息
4. 点击「**保存**」完成更新

### 删除导航

1. 找到需要删除的导航项
2. 点击「**删除**」按钮
3. 在确认弹窗中点击「**确定**」

> ⚠️ **注意**：删除后无法恢复，请谨慎操作！

### 调整顺序

1. 鼠标悬停在导航项上
2. 点击拖拽图标
3. 拖动到目标位置
4. 松开鼠标完成排序

**排序建议：**
- 重要功能放在左侧（首页、应用中心等）
- 次要功能放在右侧
- 符合用户浏览习惯

## 💡 导航设计最佳实践

### 1. 命名规范

| 推荐 ✅ | 不推荐 ❌ |
|--------|----------|
| 首页 | 主页/Home |
| 应用中心 | AI 应用/应用/工具 |
| 会员中心 | 我的账户/个人中心 |
| 定价 | 价格/收费标准 |
| 关于我们 | 公司简介/联系我们 |

### 2. 结构建议

```
标准导航结构（5-7 个）：
1. 首页
2. 应用中心
3. 解决方案
4. 定价
5. 文档
6. 关于我们
```

### 3. 用户体验

- ✅ 导航数量控制在 5-8 个
- ✅ 名称简洁易懂
- ✅ 重要功能优先展示
- ✅ 避免层级过深
- ✅ 保持视觉一致

## 📱 响应式适配

导航在不同设备上的展示：

| 设备 | 展示方式 | 说明 |
|------|----------|------|
| **桌面端** | 完整导航栏 | 显示所有导航项 |
| **平板** | 折叠菜单 | 部分导航收入菜单 |
| **手机** | 汉堡菜单 | 全部导航收入菜单 |

**设计建议：**
- 核心导航优先展示
- 次要导航可收入菜单
- 测试不同设备效果

## ❓ 常见问题

### Q1: 导航最多可以添加几个？
- 最多支持 8 个导航栏
- 建议控制在 5-7 个为佳

### Q2: 如何设置外部链接？
- 类型选择「外部链接」
- 填写完整 URL（含 https）
- 建议设置新窗口打开

### Q3: 导航不显示怎么办？
- 检查是否保存成功
- 清除缓存后刷新
- 确认前台模板支持导航

### Q4: 可以设置二级导航吗？
- 当前版本不支持二级导航
- 可使用下拉菜单替代
