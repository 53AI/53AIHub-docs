# 🎨 模板风格

53AI-Hub 内预设了基础的两种站点风格模板，你可以按照自己的需求，通过选择**站点风格**和**主题配色**自定义站点外观，打造符合品牌调性的展示效果。

![模板风格配置](http://kmdev.53ai.com/api/preview/template-style.png)

---

## 📦 功能概览

| 功能 | 说明 |
|------|------|
| **实时预览** | 调整配置时实时展示效果 |
| **站点风格** | 选择网站风格或软件风格 |
| **主题配色** | 自定义各元素颜色 |
| **模板切换** | 切换预设页面模板 |

## 👁️ 预览

在调整站点风格和颜色搭配时，系统将实时展示预览效果，便于你直观地查看修改结果，提升配置效率。

**预览区域说明：**
- 左侧：配置面板
- 右侧：实时预览
- 顶部：切换设备视图（桌面/平板/手机）

## 🖼️ 站点风格

目前支持两种站点风格，你可根据站点用途与目标用户，按需求选择风格类型。

### 1. 网站风格

**特点：**
- 功能模块丰富
- 页面结构清晰
- 更丰富的 UI 与体验层次
- 适合内容展示型站点

**适用场景：**
- 企业官网
- 产品展示
- 门户站点
- 内容平台

![网站风格示例](http://kmdev.53ai.com/api/preview/website-style.png)

### 2. 软件风格

**特点：**
- 简洁高效的界面
- 直接简约的设计
- 适合频繁的功能操作
- 适合持续使用的应用

**适用场景：**
- SaaS 应用
- 工具型产品
- 后台系统
- 效率应用

![软件风格示例](http://kmdev.53ai.com/api/preview/software-style.png)

## 🎨 主题配色

支持对站点内以下元素进行自定义颜色：

### 可配置元素

| 元素 | 说明 | 建议 |
|------|------|------|
| **主题颜色** | 站点主要按钮、选中状态等关键元素颜色 | 使用品牌主色 |
| **正文文本颜色** | 站点主要文字的颜色 | 深灰色 (#333) 易读 |
| **导航背景** | 顶部导航栏的背景色 | 与主题色协调 |
| **导航文字颜色** | 顶部导航栏的文字颜色 | 确保对比度足够 |
| **页脚背景** | 底部信息区域的背景色 | 深色或浅灰色 |
| **页脚文字颜色** | 底部信息区域的文字颜色 | 与背景形成对比 |

### 配色建议

**科技风格：**
```
主题色：#1890FF (蓝色)
文字：#333333
导航：#FFFFFF
页脚：#001529
```

**商务风格：**
```
主题色：#1E40AF (深蓝)
文字：#1F2937
导航：#F3F4F6
页脚：#111827
```

**活力风格：**
```
主题色：#F59E0B (橙色)
文字：#1F2937
导航：#FFFFFF
页脚：#374151
```

**简约风格：**
```
主题色：#10B981 (绿色)
文字：#111827
导航：#FFFFFF
页脚：#F9FAFB
```

## 📋 配置步骤

1. **选择站点风格**
   - 点击「网站风格」或「软件风格」
   - 预览区域实时更新

2. **配置主题颜色**
   - 点击颜色选择器
   - 选择或输入色值
   - 预览效果

3. **调整其他元素**
   - 依次配置各项颜色
   - 确保整体协调

4. **保存设置**
   - 确认预览效果满意
   - 点击「保存」按钮
   - 配置立即生效

## 💡 设计最佳实践

### 1. 色彩搭配原则

- **60-30-10 法则**：主色 60%，辅助色 30%，强调色 10%
- **对比度**：确保文字与背景有足够对比度
- **一致性**：全站保持统一的色彩体系
- **品牌色**：优先使用品牌标准色

### 2. 无障碍设计

| 要求 | 说明 |
|------|------|
| **文字对比度** | 至少 4.5:1（WCAG AA 标准） |
| **颜色不唯一** | 不用颜色单独传达信息 |
| **色盲友好** | 避免红绿搭配 |

### 3. 响应式适配

- 测试不同设备上的颜色效果
- 确保移动端可读性
- 考虑深色模式适配

## 🎯 页面模板

用户可为站点选择一个预设模板样式。

### 可选模板

| 模板名称 | 说明 | 适用场景 |
|----------|------|----------|
| **AI 门户** | 功能模块丰富，适合展示多个分类入口和智能体推荐内容 | 综合型 AI 平台 |
| **豆包风格** | 结构紧凑、轻量简洁，适合少量核心功能的场景 | 轻量级应用 |
| **秘塔风格** | 网格式排列，多工具集合展示，适合企业入口聚合 | 工具集合站 |
| **Kimi 风格** | 更具沉浸感的居中式内容排布，适合聚焦型应用 | 单核心应用 |
| **独立站** | 页面结构高度自由，适合做品牌型站点或嵌入外链使用 | 品牌官网 |

点击「**查看示例**」可预览模板效果，点击「**编辑**」可进一步修改页面模块样式。

## ❓ 常见问题

### Q1: 修改后多久生效？
- 保存后立即生效
- 前台可能需要刷新查看

### Q2: 可以自定义 CSS 吗？
- 可在「三方统计」中注入自定义 CSS
- 建议先备份原配置

### Q3: 如何恢复默认配色？
- 重新选择默认颜色值
- 或联系技术支持重置

### Q4: 支持深色模式吗？
- 当前版本需手动配置深色主题
- 自动深色模式开发中
