# 👥 注册用户

在「**注册用户**」界面，可以查看用户的数据，包括用户名、手机号码、电子邮箱、订阅方式、服务期限、注册时间等信息。

![注册用户列表](http://kmdev.53ai.com/api/preview/user-list.png)

---

## 📦 功能概览

| 功能 | 说明 |
|------|------|
| **用户列表** | 查看所有注册用户信息 |
| **信息编辑** | 修改用户基本信息和订阅 |
| **订阅管理** | 调整用户订阅类型和期限 |
| **用户删除** | 删除用户账号 |
| **搜索筛选** | 快速定位特定用户 |

---

## 📋 用户信息

用户列表展示以下信息：

| 字段 | 说明 |
|------|------|
| **用户名** | 用户的展示名称 |
| **手机号码** | 绑定的手机号 |
| **电子邮箱** | 绑定的邮箱地址 |
| **订阅方式** | 当前订阅类型（免费/黄金/钻石） |
| **服务期限** | 订阅服务到期时间 |
| **注册时间** | 用户账号创建时间 |
| **最后登录** | 最近一次登录时间 |

---

## ✏️ 用户信息编辑

点击用户右侧的「**编辑**」按钮，可以对该用户的基本信息进行设置与修改。

![编辑用户](http://kmdev.53ai.com/api/preview/edit-user.png)

### 可编辑字段

| 字段 | 说明 | 要求 |
|------|------|------|
| **头像** | 点击"头像"区域，可对用户的头像进行修改 | PNG/JPG，<2MB |
| **用户** | 输入或修改用户名称 | 建议在 20 字以内 |
| **登录密码** | 输入内容可修改用户登录密码 | 保留空白则密码不变更 |
| **订阅** | 查看与修改用户订阅类型 | 注册用户/黄金会员/钻石会员 |
| **服务期至** | 查看与修改用户的服务期限 | 可设定具体日期 |
| **已绑定的账户** | 查看用户绑定的手机号、邮箱等 | 不可修改 |

### 操作步骤

1. 找到目标用户
2. 点击右侧「**编辑**」按钮
3. 修改需要变更的信息
4. 点击「**保存**」完成更新

---

## 🗑️ 删除用户

点击用户右侧的「**删除**」按钮，可对该用户及现有的基本信息进行删除。

### 操作步骤

1. 找到需要删除的用户
2. 点击右侧「**删除**」按钮
3. 在确认弹窗中点击「**确定**」

> ⚠️ **警告**：
> - 删除后用户数据无法恢复
> - 用户的订单记录将保留
> - 请谨慎操作！

## 🔍 搜索与筛选

### 搜索用户

在搜索框中输入以下信息进行快速查找：
- 用户名
- 手机号码
- 电子邮箱
- 用户 ID

### 筛选用户

可按以下条件筛选用户：
- 订阅类型（免费/黄金/钻石）
- 注册时间范围
- 服务状态（正常/过期）

## ❓ 常见问题

### Q1: 如何重置用户密码？
- 进入用户编辑页面
- 在「登录密码」字段输入新密码
- 保存后生效

### Q2: 用户忘记密码怎么办？
- 指导用户使用「忘记密码」功能
- 或通过编辑功能手动重置

### Q3: 如何延长用户服务期？
- 点击用户编辑
- 修改「服务期至」字段
- 选择新的到期日期

### Q4: 删除用户后数据会怎样？
- 用户账号被删除
- 订单记录保留
- 无法恢复账号
