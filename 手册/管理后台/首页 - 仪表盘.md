# 📊 首页/仪表盘

**首页**是管理后台的默认入口页面，提供站点运营数据的概览视图和快捷操作入口，帮助管理员快速掌握站点运行状态。

![首页仪表盘](http://kmdev.53ai.com/api/preview/dashboard-overview.png)

---

## 📈 数据概览

首页顶部展示核心运营指标卡片，包含以下关键数据：

| 指标 | 说明 |
|------|------|
| **总用户数** | 站点累计注册用户数量 |
| **活跃用户** | 近期（7 天/30 天）有操作行为的用户数 |
| **订单总数** | 累计完成的订单数量 |
| **订阅收入** | 订阅服务产生的总收入（按选定币种显示） |
| **智能体调用量** | 前台用户调用智能体的总次数 |
| **API 调用量** | 通过 API 接入的调用次数统计 |

> 💡 **提示**：点击数据卡片可查看详细趋势图表和数据分析报告

---

## 🎯 快捷操作

首页提供常用功能的快捷入口，提升管理效率：

### 常用快捷方式

| 快捷操作 | 功能说明 | 跳转位置 |
|----------|----------|----------|
| ➕ **添加智能体** | 快速接入新的智能体应用 | 应用管理 → 智能体 |
| 📝 **添加提示词** | 创建新的提示词模板 | 应用管理 → 提示词 |
| 🎨 **站点装修** | 自定义站点外观 | 站点装修 → 模板风格 |
| 💰 **订单管理** | 查看和处理订单 | 运营管理 → 订单数据 |
| 👥 **用户管理** | 管理注册用户 | 运营管理 → 注册用户 |
| ⚙️ **平台接入** | 配置第三方平台 | 站点配置 → 平台接入 |

---

## 📊 动态信息区

### 最近活动
展示站点近期的关键操作记录：
- 新用户注册通知
- 订单支付成功提醒
- 智能体调用异常告警
- 系统更新公告

### 待办事项
提醒需要管理员处理的事项：
- 待确认的手动转账订单
- 待审核的用户申请
- 即将到期的订阅服务
- 需要更新的 SSL 证书

---

## 🔧 使用技巧

### 1. 数据刷新
- 页面数据默认每 **5 分钟** 自动刷新
- 点击右上角 🔄 按钮可手动刷新
- 支持选择时间范围查看历史数据

### 2. 快捷导航
- 使用左侧菜单栏快速切换模块
- 支持收藏常用页面到快捷栏
- 可使用搜索功能快速定位功能

### 3. 数据导出
- 点击数据卡片右上角 📥 可导出 Excel/CSV 格式报表
- 支持设置定时报告发送到邮箱

---

## ⚠️ 注意事项

1. **数据延迟**：部分统计数据可能存在 5-10 分钟延迟
2. **权限限制**：不同管理员角色看到的仪表盘内容可能不同
3. **浏览器兼容**：建议使用 Chrome、Edge 等现代浏览器访问
